DOCKER/PODMAN CLIENT CONFIGURATION:
----------------------------------
To pull/push images from your local machine, you need to configure it to allow
insecure registry access:

For Docker:
  Edit /etc/docker/daemon.json (create if doesn't exist):
  {
    "insecure-registries": ["132.186.17.22:5000"]
  }
  Then restart: sudo systemctl restart docker

For Podman:
  Add to /etc/containers/registries.conf:
  [[registry]]
  location = "132.186.17.22:5000"
  insecure = true

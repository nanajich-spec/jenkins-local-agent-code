Step 1: Configure Docker/Podman to Trust the Registry
──────────────────────────────────────────────────────

FOR DOCKER (on your local machine):
────────────────────────────────────
1. Edit or create: /etc/docker/daemon.json

   sudo nano /etc/docker/daemon.json

2. Add this content:

   {
     "insecure-registries": ["132.186.17.22:5000"]
   }

3. Save and restart Docker:

   sudo systemctl restart docker

   # On Mac/Windows Docker Desktop:
   # Go to Settings → Docker Engine → Add to JSON → Apply & Restart


FOR PODMAN (on your local machine):
────────────────────────────────────
1. Edit: /etc/containers/registries.conf

   sudo nano /etc/containers/registries.conf

2. Add to the end:

   [[registry]]
   location = "132.186.17.22:5000"
   insecure = true

3. No restart needed!

📦 HOW TO PUSH IMAGES TO THE REGISTRY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

METHOD 1: Build and Push New Image
───────────────────────────────────

# Build your image with registry URL in the name
docker build -t 132.186.17.22:5000/catool:1-0-0-beta .

# Or with Podman
podman build -t 132.186.17.22:5000/catool:1-0-0-beta .

# Push to registry
docker push 132.186.17.22:5000/catool:1-0-0-beta
# OR
podman push 132.186.17.22:5000/catool:1-0-0-beta --tls-verify=false


METHOD 2: Tag Existing Local Image
───────────────────────────────────

# List your local images
docker images
# OR
podman images

# Tag an existing image
docker tag myapp:latest 132.186.17.22:5000/myapp:latest
# OR
podman tag myapp:latest 132.186.17.22:5000/myapp:latest

# Push to registry
docker push 132.186.17.22:5000/myapp:latest
# OR
podman push 132.186.17.22:5000/myapp:latest --tls-verify=false


METHOD 3: Pull from Another Registry and Re-push
─────────────────────────────────────────────────

# Pull from Docker Hub
docker pull nginx:latest

# Tag for your local registry
docker tag nginx:latest 132.186.17.22:5000/nginx:latest

# Push to your registry
docker push 132.186.17.22:5000/nginx:latest


METHOD 4: Load from TAR Archive
────────────────────────────────

# If you have image as tar file
docker load -i myimage.tar
# OR
podman load -i myimage.tar

# Find the loaded image ID
docker images

# Tag it
docker tag <IMAGE_ID> 132.186.17.22:5000/myapp:version

# Push it
docker push 132.186.17.22:5000/myapp:version
